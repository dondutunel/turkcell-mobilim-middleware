# turkcell-mobilim-middleware

## Installation

#### With Docker

- Install Docker
- Run the following commands

```
docker build . -t turkcell/middleware
docker run -p 80:80 turkcell/middleware
```

#### Without Docker

- Install node v10
- Run the following commands

```
npm i
npm run start
```

## Usage

- Go to the following link through your browser

```
localhost:80
```
